Browsersync Demo
====================

「デザイナーさんでも分かる Browsersync からはじめる作業効率化」デモ。

## Usage

node.js は別途インストールしてください。

1. ファイル一式を clone かダウンロード
2. ターミナル等で該当フォルダに移動して以下のコマンドを実行  
$ npm install
3. Browsersync を起動  
$ npm start
4. ブラウザが立ち上がって localhost:3000 が表示されればOK
5. html、cssファイルを編集して Browsersync の動作を確認
